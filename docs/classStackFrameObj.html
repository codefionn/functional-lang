Detailed Description

template<typename T>
class StackFrameObj< T >

Class for adding Expr's to stack frame of Environment.

Should only be created in stack not heap. The class adds an expression to stack frame on initialization or assignment and removes the expression from stack frame on deconstruction or if new expression assigned.
